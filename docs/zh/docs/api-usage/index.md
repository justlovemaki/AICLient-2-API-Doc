# API 使用

AIClient-2-API 提供了一个统一的 API 服务，旨在兼容 OpenAI、Gemini 和 Claude 等主流 LLM 的 API 格式。这意味着你可以使用熟悉的 OpenAI SDK 或其他客户端，向 AIClient-2-API API 发送请求，而 AIClient-2-API 会在后端自动进行适配和转发。
