# API Usage

AIClient-2-API provides a unified API service designed to be compatible with mainstream LLM API formats such as OpenAI, Gemini, and Claude. This means you can use familiar OpenAI SDKs or other clients to send requests to the AIClient-2-API, and AIClient-2-API will automatically adapt and forward them on the backend.
